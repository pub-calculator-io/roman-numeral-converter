Roman numeral converter produces Roman numerals from Arabic numbers or converts Arabic numbers to Roman numerals, with values between 1 and 3,999,999.

[https://www.calculator.io/roman-numeral-converter/](https://www.calculator.io/roman-numeral-converter/)

== Usage ==

Add the Roman Numeral Converter shortcode to your page, post or sidebar:

`[ci_roman_numeral_converter]`

Add the following code to your template where you would like the Roman Numeral Converter to appear:

`<?php display_ci_roman_numeral_converter(); ?>`

== Installation ==

1. Upload the Roman Numeral Converter /ci_roman_numeral_converter/ folder to the /wp-content/plugins/ directory.
2. Activate the Roman Numeral Converter plugin through the "Plugins" menu in WordPress.
